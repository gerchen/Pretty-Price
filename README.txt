"Pretty Price" extension rounds the prices ending in 99 off to integers in the popular online shops - www.amazon.com, www.amazon.co.uk, www.amazon.de, more shops are expected in the next versions. It helps you to see real prices and protects you from "99" tricks. Below you can see the examples of how "pretty" prices will look like:

$1,499.99   ->    $1,500.00
$939.99     ->    $940.00
$19.99      ->    $20.00
$2,229.90   ->    $2,230.00
$89.99      ->    $90.00
$849.00     ->    $850.00

Just try it, it is really useful helper :)
